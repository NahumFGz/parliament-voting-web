# Instalaciones iniciales

Importante: Este proyecto usa node22.12.0

- Vite & react
  npm create vite@latest

- Tailwind
  https://tailwindcss.com/docs/guides/vite

- React router dom
  npm install react-router-dom

- Toast
  npm install sonner

# Vite

npm create vite@latest
npm install
npm run dev

# PdfToImage

sudo apt update && sudo apt install poppler-utils -y
